# Random Color
## MiniProject javascript

Color Flipper คือ เว็บไซต์สำหรับการสุ่มค่าสี

- กดปุ่ม New Color! เพื่อทำการสุ่มค่าสี
- ✨Magic ✨

## Features

- สามารมสุ่มค่าสีออกมาเป็นเลขฐานสิบหกทั้งหมด 6 ตัวเพื่อที่จะสามารถนำค่าสีไปใช้ในการตกแต่ง
